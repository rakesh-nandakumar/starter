# API Documentation - Version 103

## Endpoints (Version 103)

### GET /api/users/103
Returns list of users

### POST /api/users/103
Creates new user

### PUT /api/users/:id/103
Updates existing user

### DELETE /api/users/:id/103
Deletes user
