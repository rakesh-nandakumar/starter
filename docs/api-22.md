# API Documentation - Version 22
Last updated: 2025-06-19-16-00

## Endpoints (Version 22)

### GET /api/users/22
Returns list of users

### POST /api/users/22
Creates new user

### PUT /api/users/:id/22
Updates existing user

### DELETE /api/users/:id/22
Deletes user

## Authentication
All endpoints require valid API key.
