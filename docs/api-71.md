# API Documentation - Version 71
Last updated: 2025-06-19-16-00

## Endpoints (Version 71)

### GET /api/users/71
Returns list of users

### POST /api/users/71
Creates new user

### PUT /api/users/:id/71
Updates existing user

### DELETE /api/users/:id/71
Deletes user

## Authentication
All endpoints require valid API key.
