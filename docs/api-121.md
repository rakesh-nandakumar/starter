# API Documentation - Version 121
Last updated: 2025-06-19-16-09

## Endpoints (Version 121)

### GET /api/users/121
Returns list of users

### POST /api/users/121
Creates new user

### PUT /api/users/:id/121
Updates existing user

### DELETE /api/users/:id/121
Deletes user

## Authentication
All endpoints require valid API key.
