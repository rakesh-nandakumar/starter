# API Documentation - Version 101
Last updated: 2025-06-19-15-31

## Endpoints (Version 101)

### GET /api/users/101
Returns list of users

### POST /api/users/101
Creates new user

### PUT /api/users/:id/101
Updates existing user

### DELETE /api/users/:id/101
Deletes user

## Authentication
All endpoints require valid API key.
