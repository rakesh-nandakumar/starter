# API Documentation - Version 2091

## Endpoints (Version 2091)

### GET /api/users/2091
Returns list of users

### POST /api/users/2091
Creates new user

### PUT /api/users/:id/2091
Updates existing user

### DELETE /api/users/:id/2091
Deletes user

## Authentication
