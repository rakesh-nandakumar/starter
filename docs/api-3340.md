# API Documentation - Version 3340

## Endpoints (Version 3340)

### GET /api/users/3340
Returns list of users

### POST /api/users/3340
Creates new user

### PUT /api/users/:id/3340
Updates existing user

### DELETE /api/users/:id/3340
Deletes user

## Authentication
