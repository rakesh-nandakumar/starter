# API Documentation - Version 742

## Endpoints (Version 742)

### GET /api/users/742
Returns list of users

### POST /api/users/742
Creates new user

### PUT /api/users/:id/742
Updates existing user

### DELETE /api/users/:id/742
Deletes user
