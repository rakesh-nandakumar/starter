# API Documentation - Version 701

## Endpoints (Version 701)

### GET /api/users/701
Returns list of users

### POST /api/users/701
Creates new user

### PUT /api/users/:id/701
Updates existing user

### DELETE /api/users/:id/701
Deletes user

## Authentication
