# API Documentation - Version 262

## Endpoints (Version 262)

### GET /api/users/262
Returns list of users

### POST /api/users/262
Creates new user

### PUT /api/users/:id/262
Updates existing user

### DELETE /api/users/:id/262
Deletes user

## Authentication
