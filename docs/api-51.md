# API Documentation - Version 51
Last updated: 2025-06-19-15-30

## Endpoints (Version 51)

### GET /api/users/51
Returns list of users

### POST /api/users/51
Creates new user

### PUT /api/users/:id/51
Updates existing user

### DELETE /api/users/:id/51
Deletes user

## Authentication
All endpoints require valid API key.
