# API Documentation - Version 1332

## Endpoints (Version 1332)

### GET /api/users/1332
Returns list of users

### POST /api/users/1332
Creates new user

### PUT /api/users/:id/1332
Updates existing user

### DELETE /api/users/:id/1332
Deletes user
