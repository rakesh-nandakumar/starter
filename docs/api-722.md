# API Documentation - Version 722
Last updated: 2025-06-19-15-32

## Endpoints (Version 722)

### GET /api/users/722
Returns list of users

### POST /api/users/722
Creates new user

### PUT /api/users/:id/722
Updates existing user

### DELETE /api/users/:id/722
Deletes user

## Authentication
All endpoints require valid API key.
