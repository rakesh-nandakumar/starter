# API Documentation - Version 812

## Endpoints (Version 812)

### GET /api/users/812
Returns list of users

### POST /api/users/812
Creates new user

### PUT /api/users/:id/812
Updates existing user

### DELETE /api/users/:id/812
Deletes user

## Authentication
