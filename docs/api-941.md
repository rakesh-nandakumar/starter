# API Documentation - Version 941

## Endpoints (Version 941)

### GET /api/users/941
Returns list of users

### POST /api/users/941
Creates new user

### PUT /api/users/:id/941
Updates existing user

### DELETE /api/users/:id/941
Deletes user

## Authentication
