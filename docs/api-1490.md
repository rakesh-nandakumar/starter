# API Documentation - Version 1490

## Endpoints (Version 1490)

### GET /api/users/1490
Returns list of users

### POST /api/users/1490
Creates new user

### PUT /api/users/:id/1490
Updates existing user

### DELETE /api/users/:id/1490
Deletes user

## Authentication
