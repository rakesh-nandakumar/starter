# API Documentation - Version 750
Last updated: 2025-06-19-16-16

## Endpoints (Version 750)

### GET /api/users/750
Returns list of users

### POST /api/users/750
Creates new user

### PUT /api/users/:id/750
Updates existing user

### DELETE /api/users/:id/750
Deletes user

## Authentication
All endpoints require valid API key.
