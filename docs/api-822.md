# API Documentation - Version 822
Last updated: 2025-06-19-16-17

## Endpoints (Version 822)

### GET /api/users/822
Returns list of users

### POST /api/users/822
Creates new user

### PUT /api/users/:id/822
Updates existing user

### DELETE /api/users/:id/822
Deletes user

## Authentication
All endpoints require valid API key.
