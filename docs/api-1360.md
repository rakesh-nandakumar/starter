# API Documentation - Version 1360

## Endpoints (Version 1360)

### GET /api/users/1360
Returns list of users

### POST /api/users/1360
Creates new user

### PUT /api/users/:id/1360
Updates existing user

### DELETE /api/users/:id/1360
Deletes user
