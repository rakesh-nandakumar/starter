# API Documentation - Version 502
Last updated: 2025-06-19-16-13

## Endpoints (Version 502)

### GET /api/users/502
Returns list of users

### POST /api/users/502
Creates new user

### PUT /api/users/:id/502
Updates existing user

### DELETE /api/users/:id/502
Deletes user

## Authentication
All endpoints require valid API key.
