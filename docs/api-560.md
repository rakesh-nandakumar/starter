# API Documentation - Version 560
Last updated: 2025-06-19-16-13

## Endpoints (Version 560)

### GET /api/users/560
Returns list of users

### POST /api/users/560
Creates new user

### PUT /api/users/:id/560
Updates existing user

### DELETE /api/users/:id/560
Deletes user

## Authentication
