# API Documentation - Version 1400

## Endpoints (Version 1400)

### GET /api/users/1400
Returns list of users

### POST /api/users/1400
Creates new user

### PUT /api/users/:id/1400
Updates existing user

### DELETE /api/users/:id/1400
Deletes user

## Authentication
