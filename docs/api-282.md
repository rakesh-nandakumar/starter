# API Documentation - Version 282

## Endpoints (Version 282)

### GET /api/users/282
Returns list of users

### POST /api/users/282
Creates new user

### PUT /api/users/:id/282
Updates existing user

### DELETE /api/users/:id/282
Deletes user
