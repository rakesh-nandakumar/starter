# API Documentation - Version 403
Last updated: 2025-06-19-15-31

## Endpoints (Version 403)

### GET /api/users/403
Returns list of users

### POST /api/users/403
Creates new user

### PUT /api/users/:id/403
Updates existing user

### DELETE /api/users/:id/403
Deletes user

## Authentication
All endpoints require valid API key.
