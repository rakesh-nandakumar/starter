# API Documentation - Version 62
Last updated: 2025-06-19-15-22

## Endpoints (Version 62)

### GET /api/users/62
Returns list of users

### POST /api/users/62
Creates new user

### PUT /api/users/:id/62
Updates existing user

### DELETE /api/users/:id/62
Deletes user

## Authentication
All endpoints require valid API key.
