# API Documentation - Version 460

## Endpoints (Version 460)

### GET /api/users/460
Returns list of users

### POST /api/users/460
Creates new user

### PUT /api/users/:id/460
Updates existing user

### DELETE /api/users/:id/460
Deletes user
